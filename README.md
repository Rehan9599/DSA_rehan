# DSA Rehan

A personal Data Structures & Algorithms practice repository in **C/C++**.

## Project Structure

This repo contains standalone practice programs:

- `arrays.c`, `arrays.cpp` – array basics and operations
- `Binary_Search.cpp` – binary search patterns
- `bItManiplation.cpp` – bit manipulation problems
- `hash.cpp` – hashing / map-based problems
- `heap.cpp` – heap and priority queue problems
- `hello.cpp` – basic starter program
- `LL.cpp` – linked list problems
- `matrix.c`, `matrix.cpp` – matrix operations
- `nsl.c`, `nsl.cpp` – likely stack/nearest-smaller-left practice
- `patterns.cpp` – pattern printing problems
- `recursionPattern.cpp`, `recursive.cpp` – recursion practice
- `slidingWindow&@pointer.cpp` – sliding window / two-pointer problems
- `sorting.cpp` – sorting algorithms
- `stackQueue.cpp`, `stackQueues.cpp` – stack and queue problems
- `strings.cpp` – string-related problems

## How to Compile & Run

### Compile a single C++ file

```bash
g++ <file>.cpp -o <file>.exe
./<file>.exe
```

Example:

```bash
g++ heap.cpp -o heap.exe
./heap.exe
```

### Compile a single C file

```bash
gcc <file>.c -o <file>.exe
./<file>.exe
```
